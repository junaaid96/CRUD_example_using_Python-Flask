Windows:
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

Linux/Mac:
```
python -m venv venv
source venv\Scripts\activate
pip install -r requirements.txt
```

Now run/execute the `waitress_server.py` file:
```
python waitress_server.py
```
